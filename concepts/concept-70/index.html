Term: conceptual schema language
Term ID: 70
Definition: formal language based on a conceptual formalism for the purpose of representing conceptual schemas
notes: Note 1 to entry: A conceptual schema language may be lexical or graphical. Several conceptual schema languages can be based on the same conceptual formalism.
examples: EXAMPLE UML, EXPRESS, IDEF1X0
