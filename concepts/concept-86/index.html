Language: eng

Term: convex hull
Term ID: 86
Definition: smallest convex set containing a given geometric object
notes: Note 1 to entry: "Smallest" is the set theoretic smallest, not an indication of a measurement. The definition can be rewritten as "the intersection of all convex sets that contain the geometric object".
