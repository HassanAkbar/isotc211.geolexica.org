Language: eng

Term: route instruction
Term ID: 389
Definition: information needed at a point along a route in a network that allows that route to be traversed
notes: Note 1 to entry: To minimize the number of instructions needed to complete a route traversal, a default instruction can be assumed at junctions without specifically associated instructions. This default is called the main-road rule.

Term ID: 389

Term ID: 389

Language: fin

Term: reittiohjeistus
Term ID: 389
Definition: verkostoon kuuluvan reitin pisteissä tarvittava tieto, jonka avulla kyseinen reitti voidaan kulkea
notes: HUOM. Jotta reitin kulkemiseen tarvittavien ohjeiden määrä voidaan pitää mahdollisimman pienenä, voidaan risteyssolmuissa, joihin ei liity erityistä ohjeistusta, soveltaa oletusohjeistusta. Oletusohjeistusta kutsutaan kulkemisen pääsäännöksi.

Language: rus

Term: путевая инструкция
Term ID: 389
Definition: информация, необходимая в определенной точке маршрута в пределах сети и позволяющая следовать по этому маршруту
notes: С целью минимизации количества инструкций, необходимых для движения по маршруту, к примыканиям может быть приписана инструкция по умолчанию, т.е. примыкания не связаны с какими-либо специальными инструкциями. Эта инструкция называется «правило главной дороги»
