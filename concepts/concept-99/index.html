Language: eng

Term: coverage
Term ID: 99
Definition: feature that acts as a function to return values from its range for any direct position within its spatial, temporal or spatiotemporal domain
notes: Note 1 to entry:: In other words, a coverage is a feature that has multiple values for each attribute type, where each direct position within the geometric representation of the feature has a single value for each attribute type.
examples: EXAMPLE Examples include a raster image, polygon overlay, or digital elevation matrix.

Language: spa

Term: cobertura
Term ID: 99
Definition: objeto geográfico que actúa como una función que devuelve valores de su rango para cualquier posición directa dentro de su dominio espacial, temporal o espacio-temporal
notes: NOTA En otras palabras, una cobertura es un objeto geográfico que tienen valores múltiples para cada tipo de atributo, donde cada posición directa de la representación geométrica del objeto geográfico tiene un valor único para cada tipo de atributo.
examples: EJEMPLO Los ejemplos incluyen una imagen ráster, una capa de polígonos o una matriz digital de elevaciones.

Term ID: 99

Language: rus

Term: покрытие
Term ID: 99
Definition: модель объекта, представляющая собой функцию, возвращающую значения из диапазона допустимых значений для каждой точки в пространственной, временнóй или пространственно-временнóй области определения
notes: Иными словами, покрытие является пространственным объектом, который имеет многочисленные значения для каждого типа атрибута, где каждая точка внутри геометрического представления пространственного объекта имеет одно и то же значение для каждого типа атрибута
examples: Примерами являются растровое изображение, полигональный оверлей или цифровая матрица высот

Language: swe

Term: domänobjekttyp
Term ID: 99
Definition: objekttyp vars objekt definierar ett rumslig, temporal eller rumstemporal domän inom vilket objektets uppgifter kan variera med positionen
notes: Anm. 1 till termpost: Termen "coverage" översätts ofta med "yttäckande representation". Det begrepp som används i standarden (ISO 19123) avser dock såväl linjetäckande, yttäckande som volymtäckande och kan vara 2D (platt på marken) eller 3D samt ha med tidsdimensionen. Anm. 2 till termpost: Rasterdata kan beskrivas av en domänobjekttyp. Anm. 3 till termpost: En instans av en domänobjekttyp är ett domänobjekt.
examples: EXAMPLE Examples include a raster image, polygon overlay, or digital elevation matrix.
