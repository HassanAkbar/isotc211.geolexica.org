Language: eng

Term: feature attribute
Term ID: 170
Definition: characteristic of a feature
notes: Note 1 to entry: A feature attribute has a name, a data type, and a value domain associated to it. A feature attribute for a feature instance also has an attribute value taken from the value domain.Note 2 to entry: In a feature catalogue, a feature attribute may include a value domain but does not specify attribute values for feature instances.
examples: EXAMPLE 1 A feature attribute named 'colour' may have an attribute value 'green' which belongs to the data type 'text'.EXAMPLE 2 A feature attribute named 'length' may have an attribute value '82.4' which belongs to the data type 'real'.0

Language: spa

Term: atributo de objeto geográfico
Term ID: 170
Definition: característica de un objeto geográfico
notes: NOTA Un atributo de objeto geográfico tiene un nombre, un tipo de dato y un dominio de valores asociado a él. Un atributo de objeto geográfico de una instancia de objeto geográfico también tiene un valor de atributo tomado del dominio de valores.NOTA 2 En un catálogo de objetos geográficos, un atributo de objeto geográfico puede incluir un dominio de valores pero no especificar valores de atributo para instancias de objetos geográficos.
examples: EJEMPLO 1 Un atributo de objeto geográfico denominado "color" puede tener un valor de atributo "verde" que pertenece al tipo de datos "texto".EJEMPLO 2 Un atributo de objeto geográfico denominado "longitud" puede tener un valor de atributo "82.4", que pertenece al tipo de datos "real".

Language: ara

Term: صفه المعلم
Term ID: 170
Definition: الخصائص المميزة للمعلم
notes: ملاحظة (1): يكون للبيانات الوصفية للمعلم اسم , ونوع بيانات ومجال قيمة ترتبط بها. كما أن البيانات الوصفية الخاصة بنظير المعلم لها أيضاً قيمة تؤخذ من مجال القيمة.ملاحظة (2): في فهرس المعالم ، قد تشتمل بيانات وصف المعلم على مجال للقيمة ولكنها لا تحدد قيمة لبيانات وصف أمثلة للمعالم.
examples: مثال (1): يمكن لصفة معلم اسمها "لون" , أن تكون لها قيمة الصفة "أخضر" , وتكون هذه القيمة منتمية إلى نوع البيانات "نص". مثال (2): يمكن لصفة معلم اسمها "طول" أن تكون لها قيمة الصفة "82.4" , وتكون هذه القيمة منتمية إلى نوع البيانات "حقيقي".

Term: 要素属性
Term ID: 170
Definition: 要素的特征。
notes: 注1：要素的属性包括名称、数据类型及与其相关的值域。某个要素实例的要素属性也具有一个来自于其值域的属性值。注2：在要素目录中，要素属性可以包括一个值域，但并不指定要素实例的属性值。
examples: 示例1：一个名为“颜色”的要素属性有一个属性值为“绿色”，其数据类型为“文本”。示例2：一个名为“长度”的要素属性有一个属性值为“82.4”，其数据类型为“实型”。

Term: featureattribut
Term ID: 170
Definition: egenskab ved en feature
notes: NOTE 1 – En featureattribut er forbundet med et navn, en datatype og et sæt domæneværdier. examples: EKSEMPEL 1 - En featureattribut kaldet "farve" kan have attributværdien "grøn", der har datatypen "tekst".EKSEMPEL 2 - En featureattribut kaldet "længde" kan have attributværdien "82,3", der har datatypen ”real”.

Term: kohteen ominaisuus
Term ID: 170
Definition: kohteelle ominainen piirre
notes: HUOM. 1. Kohteen ominaisuudella on nimi, tietotyyppi ja arvojoukko. Kohteen yksittäiseen ilmentymään liittyvällä ominaisuudella on myös arvojoukkoon sisältyvä arvo.HUOM. 2. Kohdeluettelossa esiintyvällä ominaisuudella voi olla arvojoukko, mutta se ei määritä kohteen ilmentymien ominaisuuden arvoja.
examples: ESIM. 1. Kohteen ominaisuudella "väri" voi olla arvo "vihreä", jonka tietotyyppi on "teksti".ESIM. 2. Kohteen ominaisuudella "pituus" voi olla arvo "84.2", jonka tietotyyppi on "reaaliluku".

Language: kor

Term: 지형지물 속성
Term ID: 170
Definition: 피처의 특징
notes: 비고 1: 피처 속성은 명칭, 데이터 유형, 관련된 영역의 데이터 유형을 갖는다. 피처 인스턴스를 위한 피처속성 또한 값 영역으로부터의 속성 값을 갖는다. 비고 2: 지형지물 목록에서, 피처 속성은 값 영역을 포함하나 피처 인스턴스에 대한 속성값을 구체적으로 정의하지는 않는다.
examples: 보기 1: "색상"이란 명칭을 가진 피처 속성은 그 속성 값으로 "녹색"을 가질 수 있으며 데이터 유형은 "문자"이다. 보기 2: "길이"란 명칭을 가진 피처 속성은 그 속성 값으로 "82.4"를 가질 수 있으며 데이터 유형은 "실수"이다.

Term: атрибут (пространственного объекта)
