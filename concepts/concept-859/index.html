Language: eng

Term: hierarchical register
Term ID: 859
Definition: structured set of registers for a domain of register items, composed of a principal register and a set of subregisters
notes:
examples: EXAMPLE ISO 6523 is associated with a hierarchical register. The principal register contains organization identifier schemes and each subregister contains a set of organization identifiers that comply with a single organization identifier scheme.

Term ID: 859
Definition: 针对注册项的域建立的结构化的注册表集，由一个主注册表和一系列子注册表组成。

Term: иерерхический реестр
Term ID: 859
Definition: структурированный набор реестров для некоторого множества учетных единиц, образованный главным реестром и множеством подреестров
