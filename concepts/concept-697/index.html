Language: eng

Term: schema document
Term ID: 697
Definition: XML document containing schema component definitions and declarations
notes: Note 1 to entry: The W3C XML Schema provides an XML interchange format for schema information. A single schema document provides descriptions of components associated with a single XML namespace, but several documents may describe components in the same schema, i.e. the same target namespace.

Term: documentación de esquema
Term ID: 697
Definition: documento XML que contiene definiciones y declaraciones de los componentes de un esquema
