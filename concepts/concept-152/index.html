Term: encoding rule
Term ID: 152
Definition: identifiable collection of conversion rules that define the encoding for a particular data structure
notes: Note 1 to entry: An encoding rule specifies the types of data to be converted as well as the syntax, structure and codes used in the resulting data structure.
examples: EXAMPLE XML, ISO 10303-21, ISO/IEC 8211.0
