Term: version (temporal)
Term ID: 746
Definition: complete representation of an object at a given instance in time
notes: Note 1 to entry: Temporal versions differ from samples in that a complete description is required. In this sense a version is a complete sample able to be considered outside the domain of the temporal sequence to which it may belong.

Term ID: 746

Term: версия (временная)
Term ID: 746
Definition: полное представление объекта в некоторый момент времени
notes: Временные версии отличаются от образцов тем, что для них требуется завершенное описание. В этом смысле версия является завершенным образцом, который возможно рассматривать за пределами области временной последовательности, к которой он может принадлежать
