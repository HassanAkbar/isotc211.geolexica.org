Language: eng

Term: operation
Term ID: 319
Definition: service that can be requested from an object to affect behaviour
notes: Note 1 to entry: An operation has a signature, which may restrict the actual parameters that are possible.Note 2 to entry: Definition from UML Reference Manual: A specification of a transformation or query that an object may be called to execute.Note 3 to entry: An operation has a name and a list of parameters. A method is a procedure that implements an operation. It has an algorithm or procedure description.

Term: operación
Term ID: 319
Definition: servicio que puede ser solicitada por un objeto para afectar a su comportamento
notes: NOTA 1 Una operación tiene una especificación (firma) que puede restringir los parámetros que son realmente posibles.NOTA 2 Definicón del Manual de Referencia de UML: Una especificación de la transformación o consulta que un objeto puede llamar para ejecutar.NOTE 3 An operation has a name and a list of parameters. A method is a procedure that implements an operation. It has an algorithm or procedure description.

Language: fin

Term: operaatio
Term ID: 319
Definition: palvelu, jota voidaan pyytää objektilta käyttäytymiseen vaikuttamiseksi
notes: HUOM. 1. Operaatiolla on kutsumuoto (signature), joka saattaa rajoittaa todellisuudessa mahdollisia muuttujia.HUOM. 2. UML Reference Manual -käsikirjan määritelmä: sellaisen muunnoksen tai kyselyn määrittely, joka objekti voidaan käskeä suorittamaan.HUOM. 3. Operaatiolla on nimi ja lista muuttujia. Metodi on operaation toimeenpaneva menettely sisältäen algoritmin tai menetelmäkuvauksen.

Language: rus

Term: операция
Term ID: 319
Definition: услуга, которая может быть запрошена у некоторого объекта с целью воздействия на его поведение
notes: Операция имеет сигнатуру, которая может ограничивать возможные реальные параметрыОпределение из UML Reference Manual: "Спецификация преобразования или запроса, исполнение которого может быть затребовано от объекта" Операция имеет имя и перечень параметров. Метод является реализацией операции. Описывает алгоритм или процедуру, которая формирует результаты операции
