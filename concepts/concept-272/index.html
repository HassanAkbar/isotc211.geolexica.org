Term: linear reference system
Term ID: 272
Definition: reference system that identifies a location by reference to a segment of a linear geographic feature and distance along that segment from a given point
notes: Note 1 to entry: Linear reference systems are widely used in transportation, for example highway names and mile or kilometre markers.
