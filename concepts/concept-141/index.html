Language: eng

Term: discrete coverage
Term ID: 141
Definition: coverage that returns the same feature attribute values for every direct position within any single spatial object, temporal object, or spatiotemporal object in its domain
notes: Note 1 to entry: The domain of a discrete coverage consists of a finite set of spatial, temporal, or spatiotemporal objects.
