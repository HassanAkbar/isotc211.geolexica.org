Term: spherical coordinate system
Term ID: 713
Definition: three-dimensional coordinate system with one distance measured from the origin and two angular coordinates, commonly associated with a geodetic coordinate reference system
notes: Note 1 to entry: Not to be confused with an ellipsoidal coordinate system based on an ellipsoid 'degenerated' into a sphere.
